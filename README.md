# Tutorial Ninjas End-to-End Testing with Cypress

This project is an end-to-end testing suite for the **Tutorial Ninjas** website. We are using **Cypress** to test the core functionalities such as the **Cart**, **Login**, **Navbar**, and **Search Bar**.

## Overview

This project is designed to automate the testing of key functionalities on the Tutorial Ninjas website. It ensures that the following components work as expected:
1. **Cart**: Adding and removing items.
2. **Login**: User authentication and session management.
3. **Navbar**: Navigational links and dropdown menus.
4. **Search Bar**: Searching for products and viewing results.

## Features Tested

1. **Cart Functionality**:
   - Add items to the cart.
   - Remove items from the cart.
   - Verify the total price updates correctly.

2. **Login Functionality**:
   - User can successfully log in with valid credentials.
   - Incorrect login attempts show an appropriate error message.
   
3. **Navbar Navigation**:
   - Navigate between different pages using the navbar.
   - Check if the dropdown menus are functioning correctly.

4. **Search Bar**:
   - Search for products.
   - Validate that the correct search results are displayed.
